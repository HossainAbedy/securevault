SecureVault

Offline Password Manager for Windows

SecureVault securely stores your passwords locally using AES-256-GCM encryption, Windows DPAPI integration and browser autofill. Your data never leaves your computer.

1 Restart Firefox after installing SecureVault.
2 Open

about:addons
3 Verify that SecureVault appears in the Extensions list.
If Firefox was already running during installation, a restart is usually all that is required.

Test SecureVault

1 Launch SecureVault and unlock your vault.
2 Visit https://github.com/login
3 The floating SecureVault button should appear beside the login form.
4 Click the button to autofill your saved credentials.
5 Submitting a new login automatically offers to Save or Update credentials.

Features

Feature Description
Offline Storage Passwords remain on your PC only.
AES-256-GCM Encryption Each credential is encrypted individually.
Windows DPAPI Passwordless unlock tied to your Windows account.
Password Generator Create strong passwords instantly.
Desktop Autofill Ctrl + Shift + F anywhere in Windows.
Browser Autofill Chrome, Edge and Firefox support.
Import Passwords Import from Chrome, Edge, Firefox and CSV.

Troubleshooting

Problem Solution
Extension cannot connect Ensure SecureVault is running.
No autofill button The webpage may not contain a password field.
No Save prompt Link your vault to your Windows account.
Firefox extension missing Restart Firefox or reinstall SecureVault.
Edge extension missing Load the unpacked extension manually.
Security Notice

SecureVault never uploads your passwords or personal data. All encryption is performed locally on your computer.
